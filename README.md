# yyyyds-wild-oasis

## 简介

这是我的初始 yyyyds-wild-oasis 项目，用于 React 的学习与实践。

## 技术栈

该项目基于 Jonas Schmedtmann 的课程，使用了以下核心技术栈：

- **前端框架**: React (Vite)
- **状态管理与数据获取**: React Query (TanStack Query)
- **路由**: React Router
- **样式库**: Styled Components
- **表单管理**: React Hook Form
- **后端与数据库**: Supabase
- **图标与通知**: React Hot Toast, HiIcons
- **图表展示**: Recharts
- **日期处理**: date-fns
